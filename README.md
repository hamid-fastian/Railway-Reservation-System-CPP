# Railway Reservation System (C++)

## Overview

A console-based Railway Reservation System developed using C++ with Object-Oriented Programming (OOP) concepts and file handling. The system allows passengers to book, view, and cancel train tickets while providing an admin panel to manage trains and reservations.

## Features

### Passenger
- Register
- Login
- Search Trains
- Book Ticket
- View My Tickets
- Cancel Ticket
- Display Profile
- Update Profile

### Admin
- Login
- View All Trains
- Add Train
- Update Train
- Delete Train
- View All Reservations
- View All Users

## Technologies Used

- C++
- Object-Oriented Programming (OOP)
- File Handling
- Visual Studio

## Project Files

- `main.cpp`
- `Classes.cpp`
- `Classes.h`
- `user.txt`
- `admin.txt`
- `trains.txt`
- `reservations.txt`

## OOP Concepts Used

- Classes & Objects
- Inheritance

- Constructors
- File Handling
- CRUD Operations
- Menu-Driven Programming
